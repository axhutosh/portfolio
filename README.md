# 🧑‍💻 Ashutosh Kumar — Portfolio

A dark & sleek personal portfolio built with React + Vite. Showcasing my projects, skills, and contact info.

🔗 **Live:** [portfolio-axhutoshs-projects.vercel.app](https://portfolio-axhutoshs-projects.vercel.app/)
---

## 🛠 Tech Stack

- **Framework:** React 18
- **Bundler:** Vite
- **Styling:** CSS Modules (custom, no UI library)
- **Fonts:** Bebas Neue, DM Sans, Space Mono
- **Deployment:** Vercel

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/axhutosh/portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```

Output will be in the `dist/` folder.
